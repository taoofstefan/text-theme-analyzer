# Agent rollout — lessons from last week

Rolled out the agent workflow to a client team this week. Three takeaways: (1) the demo took longer than the build, (2) the questions were about trust, not capability, (3) the team that uses it will not be the team that built it.

I'm going to start billing for trust work separately from build work. Trust is the real product.
