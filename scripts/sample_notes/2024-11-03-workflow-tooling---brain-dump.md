# Workflow tooling — brain dump

Three things I keep wanting: a single inbox for every async agent ping, a way to pause and resume a multi-step workflow without losing context, and an audit trail I can grep six months later.

None of these exist cleanly in any of the current agent platforms. They all assume the agent runs to completion or it dies. Real consulting work is neither of those — it's ten starts, six pauses, and one finish three weeks later when the client finally approves the spec.
