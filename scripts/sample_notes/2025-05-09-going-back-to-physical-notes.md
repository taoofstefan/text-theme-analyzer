# Going back to physical notes

I archived all my old notebooks last weekend. The earliest is from 2014. The handwriting changes over time but the obsession is the same: draw the thing, write the thing, leave the thing somewhere I can find it later.

The notes app is faster and searchable but I don't remember the things I write in it. The notebooks I remember because I had to find a physical place to put them.
